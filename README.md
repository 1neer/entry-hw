# 엔트리 하드웨어 연결 프로그램(entry-hw)
entry-hw는 엔트리와 하드웨어를 연동하기 위한 프로그램 입니다. 엔트리 사이트에 접속하시면 엔트리를 사용한 블록코딩 환경과 하드웨어를 연동하는 모습을 확인해 볼 수 있습니다.

[문서페이지](https://entrylabs.github.io/docs/)
